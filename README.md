# CADVerify AI

Enterprise AI platform for engineering drawing verification.

## Repository Structure

- backend/ — API services, parsing, OCR, comparison, reporting, and storage
- frontend/ — web application for upload, review, and reporting
- docs/ — project documentation and architecture references
- samples/ — sample PDF and DXF assets
- .github/ — CI/CD and repository automation

## Status

This repository currently contains the production-ready project structure and placeholder files for future implementation.
