# CADVerify AI Frontend

Production-ready frontend foundation for the CADVerify AI platform.

## Included
- React 19 + Vite
- JavaScript
- Tailwind CSS
- React Router DOM
- Axios API client
- Reusable UI components
- Responsive dashboard layout with sidebar and topbar
- Placeholder pages for Dashboard, Compare Drawings, and Reports

## Run locally

```bash
cd frontend
npm install
npm run dev
```

The development server will start on http://localhost:3000.
